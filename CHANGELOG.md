# 日課ツール (Daily Game Launcher) v3.0.1 アップデート内容

## 新機能

### クラッシュレポート機能の実装
* 予期せぬクラッシュやエラーを `sys.excepthook` でグローバルキャッチ。
* エラー検知時に「開発者に報告しますか？」ダイアログを表示。
* 「報告する」ボタンでブラウザが開き、スタックトレース付きのGitHub Issue雛形を自動生成（APIトークン不要）。

### GitHub Issue直結フィードバックボタンの追加
* タスクトレイメニューおよび設定画面サイドバーに「💡 要望・バグ報告を送る」ボタンを追加。
* クリックでブラウザが開き、`labels=enhancement` 付きのIssue作成ページへ遷移。

### 開発環境の整備
* プロジェクトルートに `AGENTS.md` を追加。開発ルール（環境・フレームワーク・禁止事項等）を定義。

## バグ修正

* 連続起動モード使用時のタイミング競合バグを修正。(`launch_interval` の処理ロジックの境界条件を改善)

# 日課ツール (Daily Game Launcher) v2.0.0 アップデート内容

## ハイライト
### UIフルリニューアル
* PyQt6ベースの白/ライトグレー基調のフラットデザインに変更。
* 設定項目をCard UI状に整理。
* ON/OFF設定をトグルスイッチUIに変更。

### ツールチップの追加
* 各設定やボタンにマウスオーバーで簡易ヘルプを表示する機能を追加。

## システム改善
* **自動アップデート機能**: 起動時にGitHubから最新版を検知し、自動で更新プロセスを実行するアップデーターを実装。
* **専用インストーラー**: インストールおよびデスクトップショートカット作成を行う `setup.exe` の提供を開始。
* **タスクトレイ常駐**: ウィンドウを閉じてもタスクトレイ上で動作を継続する挙動に統一。
* **セキュリティの整理**: ローカル環境に依存する絶対パスのハードコードエラーを修正。
